Guide

At a glance

New here? Skim this section first. VibeStat-Text by VibeStatics is Windows R text mining software. Load interview, survey, report, paper, news, or policy text — then run frequency, word clouds, LDA, and document comparison with buttons, not scripts. Not only graduate students: also for institutions, practitioners, and classrooms. Designed by an Ed.D. from dissertation and graduate text-mining supervision to lower R entry barriers.

Who it is for

  • Researchers & graduate students — theses, dissertations, journal submissions
  • Faculty — research methods, text mining, mixed-methods labs
  • Public & private institutes — policy, evaluation, project reports, survey & interview data
  • Practice & consulting — client feedback, reviews, proposals, meeting notes
  • Content, media & UX — articles, posts, user feedback — topics, frequency, comparison
  • Anyone who needs reproducible document analysis without heavy R/Python scripting

What you do (in short)

  • Load data — PDF · Excel · HWP · Word · CSV · TXT · merge PDFs
  • Button analysis (local R) — morphology · frequency · TF-IDF · word cloud · LDA · doc compare
  • Dictionary & preprocess — stopwords, synonyms, phrases · outputs in .vibestat/output/
  • Optional AI (BYOK) — Vibe-Talk conversational assistant · Vibe-Loom draft weaving for papers & reports
  • Built-in tutorial · F1 Help — start in about five minutes

Vibe-Talk and Vibe-Loom are optional (your API key). Core text mining runs offline with built-in R — no API key required.

Why we built it · design story

VibeStat-Text was not built from a feature checklist alone. An Ed.D. scholar wrote a dissertation centered on text and document analysis through a far more manual path than this tool provides today — learning R, packages, environment setup, and scripts alone, often wanting to quit along the way.

Later, while supervising graduate students in text mining, the same barriers appeared again: time lost to installation and coding instead of corpus design, research questions, and interpretation. That field experience became the direct motive for development.

The goal is research question → corpus → analysis → interpretation with built-in R, buttons, dictionary & preprocess, and a built-in textbook. Optional Vibe-Talk and Vibe-Loom extend interpretation and drafting (your API key).

The app enables richer exploration and reproducibility than the designer had for their own dissertation — but analysis quality and interpretation remain the researcher’s responsibility. It is for research, institutions, and practice — not a student homework shortcut.

About VibeStatics · Vibe Statistics

VibeStatics (바이브스타틱스) is a platform for Vibe Statistics — academic text mining and data analysis on Windows, developed in Korea. The Korean name 바이브통계 refers to the same brand. VibeStatics goes beyond Vibe Coding (AI script generation) by offering research question → corpus → analysis → interpretation with buttons and built-in R.

Features in detail
  • Load TXT, CSV, Excel, PDF, HWP, Word; merge multiple PDFs
  • Korean morphological analysis · word & document frequency · TF-IDF
  • Word clouds · LDA topics · network graphs · document comparison
  • Dictionary manager — stopwords, synonyms, phrases; re-run analysis
  • Outputs saved under .vibestat/output/
  • Optional AI — Vibe-Talk · Vibe-Loom (your API key, BYOK)
Vibe-Talk · Vibe-Loom (optional AI)

Vibe-Talk — conversational assistant within your analysis context. Vibe-Loom — helps weave paper and report drafts (structure, citations to tables/figures). Both are optional and require your API key (BYOK).

Core text mining runs locally with built-in R — no API key. Unlike Vibe Coding (script generation), Talk and Loom extend the Vibe R workflow after you have reproducible outputs from button-driven analysis.

Beta vs formal edition

Local R analysis runs on your PC without an API key. AI features are optional (online, your key).

FAQ

Frequently asked questions

For Google and AI search — how VibeStatics differs from similar names.

What is VibeStatics?

VibeStatics is a Korean software brand for R-based text mining and statistics on Windows. VibeStat-Text is the flagship product: button-driven analysis with a built-in R engine. Also known in Korean as 바이브스타틱스 and 바이브통계 (Vibe Statistics).

Who designed VibeStat-Text and why?

An Ed.D. scholar designed it from a dissertation and graduate text-mining supervision experience. High R entry barriers — experienced firsthand and seen repeatedly in supervision — led to built-in R, buttons, and textbooks so researchers can focus on corpus, questions, and interpretation.

Full design story →

What are Vibe-Talk and Vibe-Loom?

Vibe-Talk — conversational analysis assistant. Vibe-Loom — paper/report drafting support. Optional AI (your key); core mining runs offline with built-in R.

Is VibeStatics the same as Vibe Coding or Naver VIBE?

No. Vibe Coding means generating R/Python scripts with AI. Naver VIBE is a music service. VibeStatics is independent desktop research software for corpus-based text mining.

Is it “Five Statics” (the number 5)?

No. The name is Vibe + Statistics → Vibe Statics, matching the domain vibestatics.com.

Is the English UI available?

Korean UI is complete and ready for research and teaching. The English program UI is in active development — this English web page helps global researchers find VibeStatics while the app UI is being localized.

Do I need to code in R?

Core workflows use buttons — no tm/ggplot scripts required for frequency, word clouds, LDA, or document comparison. The R engine runs locally inside the app. Advanced users can still view R code and console.

Do I need an API key?

No for core text mining — runs offline with built-in R. Optional AI (Vibe-Talk, Vibe-Loom) requires your own API key (BYOK).
