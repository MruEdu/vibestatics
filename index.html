📋 ZIP 받은 뒤 — 4단계
  1. ① ZIP 다운로드
    153 MB · 100%까지 기다리기
    Edge·Chrome ⚠ → ⋯ → 「유지」 (또는 「그래도 유지」)
  2. ② 압축 풀기
    ZIP 우클릭 → 「압축 풀기」 (바탕화면·다운로드 등)
    ZIP 안만 열지 말고, 풀린 폴더로 들어가세요.
  3. ③ VibeStat.exe 실행
    폴더 VibeStat-Text-0.7.7-beta.2-win64 안의 VibeStat.exe 더블클릭
    ⚠ 뜨면: exe 우클릭 → 속성 → 「차단 해제」 ✔ → 「추가 정보」→「실행」
  4. ④ 첫 실행
    「설치 완료」·준비 메시지 → 앱 첫 준비입니다 (Windows 설치 아님)
    시범판 이용약관 동의 → 프로그램 실행 (인증번호 불필요)
    이후에는 같은 폴더에서 VibeStat.exe만 누르면 바로 실행됩니다.
