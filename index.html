⚠ 받은 뒤 실행 방법 (ZIP 권장)

ZIP은 exe보다 Edge에서 막히는 경우가 적습니다.
학교·회사 PC는 보안 정책(WDAC)으로 실행이 안 될 수 있습니다.

  1. ① ZIP 받기 → 다운로드 100%까지 기다리기
    Edge ⚠ → ⋯ → 「유지」
  2. ② 압축 풀기
    ZIP 우클릭 → 「압축 풀기」 또는 「모두 압축 풀기」
    폴더 VibeStat-Text-0.7.7-beta.2-win64 안의 VibeStat.exe 를 실행
  3. ③ 차단 해제 → 실행
    VibeStat.exe 우클릭 → 속성 → 「차단 해제」 ✔ → 적용
    더블클릭 → 「추가 정보」 → 「실행」
