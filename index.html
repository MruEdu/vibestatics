📋 ZIP 받은 뒤 — 4단계
  1. ① ZIP 다운로드 — 153 MB · 100% · Edge ⚠ → 유지
  2. ② 압축 풀기 — ZIP 안만 열지 말고 풀린 폴더로
  3. ③ VibeStat.exe — 차단 해제 → 추가 정보 → 실행
  4. ④ 첫 실행 — 「설치 완료」= 앱 준비 · 약관 동의 → 바로 사용
