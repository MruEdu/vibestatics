⚠ 실행이 안 될 때 — 3단계만

포터블 파일만 받으세요. (설치형은 쓰지 마세요)
경고 ⚠ 는 Windows가 낯선 프로그램이라 막는 것이지, 바이러스가 아닙니다.

  1. ① 차단 해제
    「다운로드」 폴더 → VibeStat-Portable-…exe 우클릭 → 속성
    맨 아래 「보안: 차단 해제」에 ✔ → 적용 → 확인
  2. ② 실행
    파일을 더블클릭 → 파란 창이 뜨면
    「추가 정보」 클릭 → 「실행」 클릭
  3. ③ 그래도 안 되면
    학교·회사 PC는 보안 정책으로 막힐 수 있습니다. 개인 PC에서 시도하거나 IT 관리자에게 문의하세요.

GitHub에서 코드 입력 창이 뜨면 → 무시·닫기 (다운로드와 무관합니다)
