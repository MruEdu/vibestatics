Google Analytics 4 (GA4) 설정 — vibestatics.com
==========================================

■ 1. GA4 속성 만들기 (5분, 무료)

1. https://analytics.google.com 로그인 (hyc6999@gmail.com 등)
2. 「측정 시작」 또는 관리(톱니) → 「속성 만들기」
3. 속성 이름: VibeStatics
4. 보고 시간대: 대한민국
5. 비즈니스 정보: 소규모 / 소프트웨어·교육 등 선택
6. 「데이터 스트림」→ 「웹」
7. 웹사이트 URL: https://vibestatics.com
8. 스트림 이름: vibestatics.com
9. 「측정 ID」 복사 → G-XXXXXXXXXX 형태

■ 2. 랜딩에 ID 넣기

index.html 맨 아래:
  data-measurement-id="G-XXXXXXXXXX"
  ↑ 여기에 붙여넣기 → git commit & push

■ 3. 통계 보는 곳 (본인만)

https://analytics.google.com
→ VibeStatics 속성 → 보고서 → 실시간 / 획득 / 참여

권한을 다른 사람에게 주지 않으면 본인만 볼 수 있습니다.

■ 4. 추적되는 항목

- 페이지 방문 (자동)
- ZIP 다운로드 버튼 클릭 (file_download)
- 실습반 Gmail 신청 클릭 (workshop_apply)
- 상단 「다운로드」「실습반」 링크 (cta_click)

※ GitHub Release의 실제 ZIP 다운로드 완료 횟수는 GitHub Release 페이지에서 별도 확인

■ 5. ID 넣기 전

data-measurement-id="" → Analytics 비활성 (사이트 정상 동작)
