VibeStat-Text 시범판 설치 동영상 — 촬영·내레이션 대본 (최악 상황 포함)
================================================================
용도: 실습반·랜딩·HiBrain · 「영상 보면서 그대로 따라하기」
권장 길이: 12~15분 (최악 분기 모두 포함 시) · 짧은 편집본 6~8분 별도 가능
촬영 PC: Windows 10 또는 11 · Edge · 처음 받는 것처럼 **새 폴더에 다운로드**

────────────────────────────────────────────────
0. 촬영 전 준비 (감독·촬영자용)
────────────────────────────────────────────────

[환경]
□ 브라우저: Microsoft Edge (Chrome도 OK, Edge 경고 장면은 Edge로)
□ 다운로드 폴더 비우거나 `C:\Users\...\Downloads\VibeStat-데모` 새 폴더
□ 바탕화면에 예전 VibeStat 바로가기·폴더 없애기 (헷갈림 방지)
□ 화면 녹화: 1920×1080 · 마우스 커서 크게 · 확대 125% 권장 (고령·초보용)
□ 마이크: 조용한 곳 · 속도 **느리게** (1.5배 빨리 말하지 않기)

[최악 상황 재현]
□ SmartScreen: **인터넷에서 GitHub Release로 직접** ZIP 받기 (USB X)
□ 「차단 해제」: ZIP·exe **둘 다** 속성에서 체크하는 장면 **반드시** 찍기
□ Edge 「다운로드가 안전하지 않을 수 있음」 → **유지** 클릭 장면
□ 바로가기 만든 뒤 **바탕화면 아이콘** 다시 실행 → SmartScreen **한 번 더** (있으면)

[OSD 자막 규칙]
· 화면 하단 고정 · 흰 글씨 + 검은 외곽 · 한 줄 15자 내외
· 클릭 직전 0.5초 **노란 원** 또는 화살표 (편집 시)

[영상 제목 후보]
「VibeStat-Text 시범판 설치 (Windows 전체) — SmartScreen·차단해제 포함」
「[따라하기] vibestatics.com ZIP 받아서 실행하기 (최초 설치)」

3. ② 차단 해제 — ZIP (2:30 ~ 4:00) ★필수

· 속성 창 **아래쪽** 스크롤
· 「보안: 이 파일은 다른 컴퓨터에서 왔으며…」 + **「차단 해제」** 체크박스

[내레이션]
「**압축 풀기 전에** 꼭 하세요. ZIP **우클릭**, **속성**,

 이 체크가 **안 보이면** Windows 버전에 따라 없을 수 있습니다. 그때는 **다음 단계**로 가도 됩니다.」

[OSD]
② ZIP 우클릭 → 속성 → 차단 해제 ✓ → 확인
